## 开始使用

1. 添加css样式

使用框架前，请再css中开启border-box。

2. 安装 z-ui

```javascript
npm install z-ui-0-1 --save
```

3. 引入z-ui

```javascript
import {Button} from 'z-ui-0-1'
import 'z-ui-0-1/dist/index.css'

export default {
  components: {
    'z-button': Button
  }
}
```
